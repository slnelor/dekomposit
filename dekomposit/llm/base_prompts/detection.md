# Language Detection Prompt

Detect the language of the following text. Return only the language code:

- en - English
- ru - Russian
- uk - Ukrainian
- sk - Slovak
- other - unknown

Be precise:
- Ukrainian uses letters і, ї, є, ґ
- Russian uses ъ, ы, э, ё
- Slovak uses diacritics like č, š, ž, ľ, ť, ň, ď
